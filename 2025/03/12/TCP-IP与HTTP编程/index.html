TCP/IP

TCP/IP жҳҜз”ЁдәҺеӣ зү№зҪ‘ (Internet) зҡ„йҖҡдҝЎеҚҸи®®гҖӮи®Ўз®—жңәйҖҡдҝЎеҚҸи®®жҳҜеҜ№йӮЈдәӣи®Ўз®—жңәеҝ…йЎ»йҒөе®Ҳд»ҘдҫҝеҪјжӯӨйҖҡдҝЎзҡ„зҡ„и§„еҲҷзҡ„жҸҸиҝ°гҖӮ

TCP/IP жҳҜдҫӣе·ІиҝһжҺҘеӣ зү№зҪ‘зҡ„и®Ўз®—жңәиҝӣиЎҢйҖҡдҝЎзҡ„йҖҡдҝЎеҚҸи®®гҖӮ

TCP/IP жҢҮдј иҫ“жҺ§еҲ¶еҚҸи®®/зҪ‘йҷ…еҚҸи®®пјҲTransmission Control Protocol Internet ProtocolпјүгҖӮ

TCP/IP е®ҡд№үдәҶз”өеӯҗи®ҫеӨҮпјҲжҜ”еҰӮи®Ўз®—жңәпјүеҰӮдҪ•иҝһе…Ҙеӣ зү№зҪ‘пјҢд»ҘеҸҠж•°жҚ®еҰӮдҪ•еңЁе®ғд»¬д№Ӣй—ҙдј иҫ“зҡ„ж ҮеҮҶ

йҖҡиҝҮж•°жҚ®й“ҫи·ҜеұӮе’Ңдј иҫ“еұӮзҡ„ж•°жҚ®йғҪжңүйҷҗеҲ¶,еҰӮеӣҫ. еңЁдј иҫ“еұӮдёҚиғҪи¶…иҝҮTCPжңҖеӨ§жҠҘж–Үж®өй•ҝеәҰ(MSS). жҜ”еҰӮдёҖж¬ЎеҶҷе…ҘеӨ§дәҺMSSзҡ„ж•°жҚ®,еҶҚеҠ дёҠHTTPеӨҙйғЁдҝЎжҒҜ,и¶…иҝҮдәҶMSSеҗҺеңЁдј иҫ“еұӮдјҡе°ҶиҝҷдәӣдҝЎжҒҜеҲҶж®ө.

жҜҸдёӘеұӮдјҡеҠ дёҠеӨҙйғЁжҲ–еӨҙе°ҫйғЁ

TCP/IPеҚҸи®®ж ҲеұһдәҺж“ҚдҪңзі»з»ҹ(еҶ…ж ёжҖҒ)зҡ„иЎҢдёә,

еә”вҪӨзЁӢеәҸпјҲжөҸи§ҲеҷЁпјүйҖҡиҝҮи°ғвҪӨ Socket еә“пјҢжқҘе§”жүҳеҚҸи®®ж ҲвјҜдҪңгҖӮеҚҸи®®ж Ҳзҡ„дёҠеҚҠйғЁеҲҶжңүдёӨеқ—пјҢеҲҶеҲ«жҳҜиҙҹиҙЈж”¶еҸ‘ ж•°жҚ®зҡ„ TCP е’Ң UDP еҚҸи®®пјҢиҝҷдёӨдёӘдј иҫ“еҚҸи®®дјҡжҺҘеҸ—еә”вҪӨеұӮзҡ„е§”жүҳжү§вҫҸж”¶еҸ‘ж•°жҚ®зҡ„ж“ҚдҪңгҖӮ

з”ЁжҲ·жҖҒйҖҡиҝҮдҪҝз”Ёsocketеә“з»“еҗҲеҜ№еә”HTTPеҚҸи®®жҗӯе»әеә”з”ЁзЁӢеәҸ

дј иҫ“еұӮTCPеҚҸи®®

TCPж•°жҚ®еҢ…ж јејҸ

вҖӢ вҫёе…ҲпјҢжәҗз«ҜвјқеҸ·е’ҢвҪ¬ж Үз«ҜвјқеҸ·жҳҜдёҚеҸҜе°‘зҡ„пјҢеҰӮжһңжІЎжңүиҝҷдёӨдёӘз«ҜвјқеҸ·пјҢж•°жҚ®е°ұдёҚзҹҘйҒ“еә”иҜҘеҸ‘з»ҷе“ӘдёӘеә”вҪӨ.

вҖӢ жҺҘдёӢжқҘжңүеҢ…зҡ„еәҸеҸ·пјҢиҝҷдёӘжҳҜдёәдәҶи§ЈеҶіеҢ…д№ұеәҸзҡ„й—®йўҳгҖӮ иҝҳжңүеә”иҜҘжңүзҡ„жҳҜзЎ®и®ӨеҸ·пјҢвҪ¬зҡ„жҳҜзЎ®и®ӨеҸ‘еҮәеҺ»еҜ№вҪ…жҳҜеҗҰжңүж”¶еҲ°.(и¶…ж—¶йҮҚдј ,йҖҡиҝҮе®ҡж—¶еҷЁе’ҢзЎ®и®ӨеәҸеҸ·ack)

вҖӢ еҰӮжһңжІЎжңүж”¶еҲ°е°ұеә”иҜҘйҮҚж–°еҸ‘йҖҒпјҢзӣҙеҲ°йҖҒиҫҫпјҢ иҝҷдёӘжҳҜдёәдәҶи§ЈеҶідёўеҢ…зҡ„й—®йўҳгҖӮ жҺҘдёӢжқҘиҝҳжңүвјҖдәӣзҠ¶жҖҒдҪҚгҖӮдҫӢеҰӮ SYN жҳҜеҸ‘иө·вјҖдёӘиҝһжҺҘпјҢ ACK жҳҜеӣһеӨҚпјҢ RST жҳҜйҮҚж–°иҝһжҺҘпјҢ FIN жҳҜз»“жқҹиҝһжҺҘзӯүгҖӮTCP жҳҜвҫҜеҗ‘иҝһжҺҘзҡ„пјҢеӣ вҪҪеҸҢвҪ…иҰҒз»ҙжҠӨиҝһжҺҘзҡ„зҠ¶жҖҒпјҢиҝҷдәӣеёҰзҠ¶жҖҒдҪҚзҡ„еҢ…зҡ„еҸ‘йҖҒпјҢдјҡеј•иө·еҸҢвҪ…зҡ„зҠ¶жҖҒ еҸҳжӣҙгҖӮ иҝҳжңүвјҖдёӘйҮҚиҰҒзҡ„е°ұжҳҜзӘ—вјқвјӨвј©.(йқўеҗ‘иҝһжҺҘ)

вҖӢ TCP иҰҒеҒҡжөҒйҮҸжҺ§еҲ¶пјҢйҖҡдҝЎеҸҢвҪ…еҗ„еЈ°жҳҺвјҖдёӘзӘ—вјқпјҲзј“еӯҳвјӨвј©пјүпјҢж ҮиҜҶвҫғвј°еҪ“еүҚиғҪеӨҹзҡ„еӨ„зҗҶиғҪвј’пјҢеҲ«еҸ‘йҖҒзҡ„еӨӘеҝ«пјҢж’‘жӯ»жҲ‘пјҢд№ҹеҲ«еҸ‘зҡ„еӨӘж…ўпјҢйҘҝжӯ»жҲ‘гҖӮ йҷӨдәҶеҒҡжөҒйҮҸжҺ§еҲ¶д»ҘеӨ–.(жөҒйҮҸжҺ§еҲ¶,йҖҡиҝҮзӘ—еҸЈеӨ§е°Ҹзҡ„ж»‘еҠЁзӘ—еҸЈжңәеҲ¶)

вҖӢ TCPиҝҳдјҡеҒҡжӢҘеЎһжҺ§еҲ¶пјҢеҜ№дәҺзңҹжӯЈзҡ„йҖҡи·Ҝе өв»ӢдёҚе өв»ӢпјҢе®ғвҪҶиғҪдёәвј’пјҢе”ҜвјҖиғҪеҒҡзҡ„е°ұжҳҜжҺ§еҲ¶вҫғвј°пјҢд№ҹеҚіжҺ§еҲ¶еҸ‘йҖҒзҡ„йҖҹеәҰ(TCPйҮҮз”ЁдәҶеҮ з§Қз®—жі•жқҘеҠЁжҖҒең°и°ғж•ҙеҸ‘йҖҒйҖҹзҺҮд»ҘйҖӮеә”зҪ‘з»ңзҠ¶еҶө).

  • жөҒйҮҸжҺ§еҲ¶дё“жіЁдәҺеҢ№й…ҚеҸ‘йҖҒж–№дёҺжҺҘж”¶ж–№д№Ӣй—ҙзҡ„еӨ„зҗҶиғҪеҠӣпјҢйҳІжӯўеҸ‘йҖҒж–№еҸ‘йҖҒи¶…еҮәжҺҘж”¶ж–№еӨ„зҗҶиғҪеҠӣзҡ„ж•°жҚ®йҮҸгҖӮ
  • жӢҘеЎһжҺ§еҲ¶еҲҷзқҖзңјдәҺж•ҙдёӘзҪ‘з»ңзҡ„зҠ¶жҖҒпјҢиҜ•еӣҫжүҫеҲ°еҗҲйҖӮзҡ„еҸ‘йҖҒйҖҹзҺҮпјҢж—ўдёҚиҝҮеәҰеҚ з”ЁзҪ‘з»ңиө„жәҗеҜјиҮҙжӢҘеЎһпјҢд№ҹдёҚиҝҮдәҺдҝқе®ҲжөӘиҙ№еёҰе®ҪгҖӮ
  1. ж…ўеҗҜеҠЁпјҲSlow Startпјү

еҪ“дёҖдёӘTCPиҝһжҺҘејҖе§Ӣж—¶пјҢеҸ‘йҖҒж–№дёҚзҹҘйҒ“зҪ‘з»ңзҡ„зҠ¶жҖҒпјҢд№ҹдёҚзҹҘйҒ“жҺҘж”¶ж–№иғҪеӨҹжҺҘеҸ—ж•°жҚ®зҡ„йҖҹеәҰгҖӮж…ўеҗҜеҠЁйҳ¶ж®өйҖҡиҝҮжҢҮж•°еўһй•ҝзҡ„ж–№ејҸйҖҗжёҗеўһеҠ жӢҘеЎһзӘ—еҸЈпјҲCongestion Window, cwndпјүзҡ„еӨ§е°ҸпјҢд»ҘдҫҝжҺўжөӢзҪ‘з»ңзҡ„е®№йҮҸгҖӮеҲқе§Ӣж—¶пјҢcwndйҖҡеёёи®ҫзҪ®дёәдёҖдёӘе°ҸеҖјпјҲеҰӮ1дёӘжңҖеӨ§ж®өеӨ§е°ҸMSSпјүгҖӮжҜҸжҲҗеҠҹжҺҘж”¶еҲ°дёҖдёӘACKзЎ®и®ӨпјҢcwndе°ұеўһеҠ дёҖдёӘMSSгҖӮиҝҷдёӘиҝҮзЁӢжҢҒз»ӯеҲ°йҒҮеҲ°йҰ–дёӘдёўеҢ…дәӢд»¶жҲ–иҖ…иҫҫеҲ°ж…ўеҗҜеҠЁйҳҲеҖјпјҲssthreshпјүгҖӮ

  1. жӢҘеЎһйҒҝе…ҚпјҲCongestion Avoidanceпјү

дёҖж—Ұcwndзҡ„еҖјиҫҫеҲ°жҲ–и¶…иҝҮssthreshпјҢTCPиҝӣе…ҘжӢҘеЎһйҒҝе…Қйҳ¶ж®өгҖӮеңЁиҝҷдёӘйҳ¶ж®өпјҢcwndзҡ„еўһй•ҝйҖҹеәҰеҸҳзј“пјҢйҮҮз”ЁзәҝжҖ§еўһй•ҝж–№ејҸпјҢеҚіжҜҸж¬ЎеҫҖиҝ”ж—¶й—ҙпјҲRTTпјүеҸӘе…Ғи®ёcwndеўһеҠ дёҖдёӘMSS/cwndгҖӮиҝҷжңүеҠ©дәҺжӣҙзІҫз»Ҷең°жҺўжөӢзҪ‘з»ңзҡ„жңҖеӨ§еҗһеҗҗйҮҸпјҢиҖҢдёҚдјҡеғҸж…ўеҗҜеҠЁйӮЈж ·иҝ…йҖҹдҪҝзҪ‘з»ңиҝҮиҪҪгҖӮ

  1. еҝ«йҖҹйҮҚдј пјҲFast Retransmitпјү

еҝ«йҖҹйҮҚдј жңәеҲ¶з”ЁдәҺеңЁжңӘи¶…ж—¶зҡ„жғ…еҶөдёӢжЈҖжөӢеҲ°дёўеҢ…гҖӮеҰӮжһңеҸ‘йҖҒж–№иҝһз»ӯж”¶еҲ°дёүдёӘеҜ№еҗҢдёҖж•°жҚ®ж®өзҡ„йҮҚеӨҚACKпјҢиҝҷж„Ҹе‘ізқҖиҜҘж•°жҚ®ж®өеҸҜиғҪе·Із»ҸдёўеӨұпјҢеҸ‘йҖҒж–№е°Ҷз«ӢеҚійҮҚдј дёўеӨұзҡ„ж•°жҚ®ж®өпјҢиҖҢдёҚжҳҜзӯүеҫ…е®ҡж—¶еҷЁи¶…ж—¶гҖӮиҝҷж ·еҒҡеҸҜд»Ҙжӣҙеҝ«ең°жҒўеӨҚдёўеӨұзҡ„ж•°жҚ®еҢ…пјҢеҮҸе°‘зӯүеҫ…ж—¶й—ҙгҖӮ

  1. еҝ«йҖҹжҒўеӨҚпјҲFast Recoveryпјү

еҝ«йҖҹжҒўеӨҚдёҺеҝ«йҖҹйҮҚдј й…ҚеҗҲдҪҝз”ЁгҖӮеҪ“еҸ‘йҖҒж–№з”ұдәҺж”¶еҲ°дёүж¬ЎйҮҚеӨҚACKиҖҢи§ҰеҸ‘еҝ«йҖҹйҮҚдј еҗҺпјҢе®ғдјҡиҝӣе…Ҙеҝ«йҖҹжҒўеӨҚзҠ¶жҖҒгҖӮеңЁжӯӨзҠ¶жҖҒдёӢпјҢеҸ‘йҖҒж–№дёҚжү§иЎҢж…ўеҗҜеҠЁпјҢиҖҢжҳҜи°ғж•ҙssthreshеҲ°еҪ“еүҚcwndзҡ„дёҖеҚҠпјҢе№¶д»ҺйӮЈйҮҢз»§з»ӯе°қиҜ•еўһеӨ§cwndпјҢзӣҙеҲ°дёҚеҶҚж”¶еҲ°йҮҚеӨҚзҡ„ACKдёәжӯўгҖӮиҝҷжңүеҠ©дәҺеҝ«йҖҹд»ҺдёўеҢ…дёӯжҒўеӨҚпјҢиҖҢдёҚеҝ…еӣһеҲ°ж…ўеҗҜеҠЁзҠ¶жҖҒгҖӮ

иҝһжҺҘж—¶дёүж¬ЎжҸЎжүӢ

иҝҷдёӘжүҖи°“зҡ„гҖҢиҝһжҺҘгҖҚпјҢеҸӘжҳҜеҸҢвҪ…и®Ўз®—жңәвҫҘз»ҙжҠӨвјҖдёӘзҠ¶жҖҒжңәпјҢеңЁиҝһжҺҘе»әвҪҙзҡ„иҝҮзЁӢдёӯпјҢеҸҢвҪ…зҡ„зҠ¶жҖҒеҸҳеҢ–ж—¶еәҸеӣҫ

еңЁ HTTP дј иҫ“ж•°жҚ®д№ӢеүҚпјҢвҫёе…ҲйңҖиҰҒ TCP е»әвҪҙиҝһжҺҘпјҢTCP иҝһжҺҘзҡ„е»әвҪҙпјҢйҖҡеёёз§°дёәдёүж¬ЎжҸЎвјҝгҖӮ

дёүж¬ЎжҸЎвјҝвҪ¬зҡ„жҳҜдҝқиҜҒеҸҢвҪ…йғҪжңүеҸ‘йҖҒе’ҢжҺҘж”¶зҡ„иғҪвј’гҖӮ

дҪҝз”Ёnetstat -antp,жҹҘзңӢtcpиҝһжҺҘзҠ¶жҖҒ

зҪ‘з»ңеұӮIPеҚҸи®®

TCP жЁЎеқ—еңЁжү§вҫҸиҝһжҺҘгҖҒж”¶еҸ‘гҖҒж–ӯејҖзӯүеҗ„йҳ¶ж®өж“ҚдҪңж—¶пјҢйғҪйңҖиҰҒе§”жүҳ IP жЁЎеқ—е°Ҷж•°жҚ®е°ҒиЈ…жҲҗвҪ№з»ңеҢ…еҸ‘йҖҒз»ҷйҖҡдҝЎеҜ№иұЎ

еӣ дёә HTTP жҳҜз»ҸиҝҮ TCP дј иҫ“зҡ„пјҢжүҖд»ҘеңЁ IP еҢ…еӨҙзҡ„еҚҸи®®еҸ·пјҢиҰҒеЎ«еҶҷдёә TCPгҖӮ 06 пјҲвј—е…ӯиҝӣеҲ¶пјү,иЎЁзӨәеҚҸи®®дёәTCP

еңЁ Linux ж“ҚдҪңзі»з»ҹ,еҸҜд»ҘдҪҝвҪӨ route -n е‘Ҫд»ӨжҹҘзңӢеҪ“еүҚзі»з»ҹзҡ„и·Ҝз”ұиЎЁ

IPж•°жҚ®еҢ…еҢ…жӢ¬йҰ–йғЁй•ҝеәҰе’ҢжҖ»й•ҝеәҰ. жӯӨеӨ–еҪ“IPж•°жҚ®еҢ…еӨ§дәҺеә•еұӮеҚҸи®®жүҖе…Ғи®ёзҡ„еӨ§е°Ҹ(MTU)ж—¶иҝӣиЎҢеҲҶзүҮ.

IPеҲҶзүҮзҡ„дё»иҰҒзү№зӮ№пјҡ

  • ж ҮиҜҶпјҲIdentificationпјүпјҡжүҖжңүеҲҶзүҮйғҪе…ұдә«зӣёеҗҢзҡ„ж ҮиҜҶеҖјпјҢд»Ҙдҫҝзӣ®ж Үдё»жңәеҸҜд»ҘиҜҶеҲ«иҝҷдәӣеҲҶзүҮеұһдәҺеҗҢдёҖдёӘеҺҹе§Ӣж•°жҚ®жҠҘгҖӮ
  • ж Үеҝ—пјҲFlagsпјүпјҡеҢ…еҗ«дёүдёӘжҜ”зү№дҪҚпјҢе…¶дёӯдёӨдёӘз”ЁдәҺжҺ§еҲ¶жҳҜеҗҰиҝҳжңүжӣҙеӨҡзҡ„еҲҶзүҮд»ҘеҸҠжҳҜеҗҰе…Ғи®ёеҲҶзүҮгҖӮ
  • зүҮеҒҸз§»пјҲFragment OffsetпјүпјҡиЎЁзӨәиҜҘеҲҶзүҮеңЁеҺҹе§Ӣж•°жҚ®жҠҘдёӯзҡ„дҪҚзҪ®пјҢеё®еҠ©жҺҘж”¶з«ҜжҢүз…§жӯЈзЎ®зҡ„йЎәеәҸйҮҚз»„ж•°жҚ®жҠҘгҖӮ

еҲҶзүҮиҝҮзЁӢ

  1. еҪ“дёҖдёӘIPж•°жҚ®еҢ…еҮҶеӨҮеҸ‘йҖҒж—¶пјҢеҰӮжһңе…¶еӨ§е°Ҹи¶…иҝҮдәҶеҮәз«ҷжҺҘеҸЈзҡ„MTUпјҢеҲҷйңҖиҰҒиҝӣиЎҢеҲҶзүҮгҖӮ
  2. жҜҸдёӘз”ҹжҲҗзҡ„еҲҶзүҮйғҪе°ҶеҢ…еҗ«жәҗең°еқҖгҖҒзӣ®зҡ„ең°еқҖгҖҒж ҮиҜҶеӯ—ж®өе’Ңж Үеҝ—еӯ—ж®өзӯүдҝЎжҒҜгҖӮ
  3. йҷӨдәҶжңҖеҗҺдёҖдёӘеҲҶзүҮеӨ–пјҢе…¶д»–жүҖжңүеҲҶзүҮйғҪе°ҶеЎ«е……еҲ°е®ғд»¬зҡ„жңҖеӨ§й•ҝеәҰпјҢиҖҢжңҖеҗҺдёҖдёӘеҲҶзүҮеҸҜиғҪе°ҸдәҺжңҖеӨ§й•ҝеәҰгҖӮ
  4. жҺҘж”¶ж–№ж”¶еҲ°жүҖжңүеҲҶзүҮеҗҺпјҢж №жҚ®ж ҮиҜҶеӯ—ж®өгҖҒж Үеҝ—еӯ—ж®өе’ҢзүҮеҒҸз§»еӯ—ж®өйҮҚз»„еҺҹе§Ӣж•°жҚ®жҠҘ

вҪЈжҲҗдәҶ IP еӨҙйғЁд№ӢеҗҺпјҢжҺҘдёӢжқҘвҪ№з»ңеҢ…иҝҳйңҖиҰҒеңЁ IP еӨҙйғЁзҡ„еүҚвҫҜеҠ дёҠ MAC еӨҙйғЁгҖӮеңЁ MAC еҢ…еӨҙвҫҘйңҖиҰҒеҸ‘йҖҒвҪ… MAC ең°еқҖе’ҢжҺҘж”¶вҪ…вҪ¬ж Ү MAC ең°еқҖпјҢвҪӨдәҺдёӨзӮ№д№Ӣй—ҙзҡ„дј иҫ“гҖӮ вјҖиҲ¬еңЁ TCP/IP йҖҡдҝЎвҫҘпјҢMAC еҢ…еӨҙзҡ„еҚҸи®®зұ»еһӢеҸӘдҪҝвҪӨпјҡ 0800 пјҡ IP еҚҸи®® 0806 пјҡ ARP еҚҸи®®

еҪ“дёҚзҹҘйҒ“MACең°еқҖж—¶,йңҖиҰҒ ARP еҚҸи®®её®жҲ‘д»¬жүҫеҲ°и·Ҝз”ұеҷЁзҡ„ MAC ең°еқҖгҖӮ

ARP еҚҸи®®дјҡеңЁд»ҘеӨӘвҪ№дёӯд»Ҙвјҙж’ӯзҡ„еҪўејҸпјҢеҜ№д»ҘеӨӘвҪ№жүҖжңүзҡ„и®ҫеӨҮе–ҠеҮәпјҡвҖңиҝҷдёӘ IP ең°еқҖжҳҜи°Ғзҡ„пјҹиҜ·жҠҠдҪ зҡ„ MAC ең°еқҖе‘ҠиҜүжҲ‘вҖқгҖӮ з„¶еҗҺе°ұдјҡжңүвјҲеӣһзӯ”пјҡвҖңиҝҷдёӘ IP ең°еқҖжҳҜжҲ‘зҡ„пјҢжҲ‘зҡ„ MAC ең°еқҖжҳҜ XXXXвҖқгҖӮ

еҰӮжһңеҜ№вҪ…е’Ңвҫғвј°еӨ„дәҺеҗҢвјҖдёӘвјҰвҪ№дёӯпјҢйӮЈд№ҲйҖҡиҝҮдёҠвҫҜзҡ„ж“ҚдҪңе°ұеҸҜд»Ҙеҫ—еҲ°еҜ№вҪ…зҡ„ MAC ең°еқҖгҖӮз„¶еҗҺе°Ҷиҝҷ дёӘ MAC ең°еқҖеҶҷвјҠ MAC еӨҙйғЁпјҢMAC еӨҙйғЁе°ұе®ҢжҲҗдәҶгҖӮ

ж“ҚдҪңзі»з»ҹдјҡжҠҠжң¬ж¬ЎжҹҘиҜўз»“жһңж”ҫеҲ°вјҖеқ—еҸ«еҒҡ ARP зј“еӯҳзҡ„еҶ…еӯҳз©әй—ҙз•ҷзқҖд»ҘеҗҺвҪӨпјҢдёҚиҝҮзј“еӯҳзҡ„ж—¶й—ҙ е°ұвјҸеҲҶй’ҹгҖӮ д№ҹе°ұжҳҜиҜҙпјҢеңЁеҸ‘еҢ…ж—¶пјҡ е…ҲжҹҘиҜў ARP зј“еӯҳпјҢеҰӮжһңе…¶дёӯе·Із»ҸдҝқеӯҳдәҶеҜ№вҪ…зҡ„ MAC ең°еқҖпјҢе°ұдёҚйңҖиҰҒеҸ‘йҖҒ ARP жҹҘиҜўпјҢзӣҙжҺҘдҪҝвҪӨ ARP зј“еӯҳдёӯзҡ„ең°еқҖгҖӮ вҪҪеҪ“ ARP зј“еӯҳдёӯдёҚеӯҳеңЁеҜ№вҪ… MAC ең°еқҖж—¶пјҢеҲҷеҸ‘йҖҒ ARP вјҙж’ӯжҹҘиҜўгҖӮ

вҪ№з»ңеҢ…еҸӘжҳҜеӯҳж”ҫеңЁеҶ…еӯҳдёӯзҡ„вјҖдёІвјҶиҝӣеҲ¶ж•°еӯ—дҝЎжҒҜпјҢжІЎжңүеҠһжі•зӣҙжҺҘеҸ‘йҖҒз»ҷеҜ№вҪ…гҖӮеӣ жӯӨйңҖиҰҒе°Ҷж•°еӯ—дҝЎжҒҜ иҪ¬жҚўдёәз”өдҝЎеҸ·пјҢжүҚиғҪеңЁвҪ№зәҝдёҠдј иҫ“пјҢд№ҹе°ұжҳҜиҜҙпјҢиҝҷжүҚжҳҜзңҹжӯЈзҡ„ж•°жҚ®еҸ‘йҖҒиҝҮзЁӢгҖӮ иҙҹиҙЈжү§вҫҸиҝҷвјҖж“ҚдҪңзҡ„жҳҜвҪ№еҚЎпјҢиҰҒжҺ§еҲ¶вҪ№еҚЎиҝҳйңҖиҰҒйқ вҪ№еҚЎй©ұеҠЁзЁӢеәҸгҖӮ вҪ№еҚЎй©ұеҠЁиҺ·еҸ–вҪ№з»ңеҢ…д№ӢеҗҺпјҢдјҡе°Ҷе…¶еӨҚеҲ¶еҲ°вҪ№еҚЎеҶ…зҡ„зј“еӯҳеҢәдёӯпјҢжҺҘзқҖдјҡеңЁе…¶ејҖеӨҙеҠ дёҠжҠҘеӨҙе’Ңиө·е§Ӣеё§еҲҶз•Ңз¬ҰпјҢ еңЁжң«е°ҫеҠ дёҠвҪӨдәҺжЈҖжөӢй”ҷиҜҜзҡ„её§ж ЎйӘҢеәҸеҲ—

HTTP

HTTPжҳҜеә”з”ЁеұӮеҚҸи®®,зұ»дјјзҡ„еә”з”ЁеұӮеҚҸи®®жңүFTP,SFTP,TELNETзӯү. HTTPеҚҸи®®еҜ№дәҺиҜ·жұӮжҠҘж–Үе’Ңе“Қеә”жҠҘж–ҮжңүдёҚеҗҢзҡ„ж јејҸиҰҒжұӮ. HTTPзүҲжң¬жңү1.0,1.1дёҺ2.0.

RPCдёҺHTTP

  • RPCпјҲRemote Procedure CallпјҢиҝңзЁӢиҝҮзЁӢи°ғз”Ёпјү:жҳҜдёҖз§ҚеҚҸи®®пјҢе…Ғи®ёдёҖдёӘзЁӢеәҸйҖҡиҝҮзҪ‘з»ңиҜ·жұӮеҸҰдёҖдёӘең°еқҖз©әй—ҙдёӯзҡ„еӯҗзЁӢеәҸжҲ–жңҚеҠЎпјҢиҖҢж— йңҖдәҶи§Јеә•еұӮзҪ‘з»ңз»ҶиҠӮгҖӮе®ғж—ЁеңЁи®©ејҖеҸ‘иҖ…еғҸи°ғз”Ёжң¬ең°еҮҪж•°дёҖж ·и°ғз”ЁиҝңзЁӢжңҚеҠЎгҖӮ
  • HTTPпјҲHypertext Transfer ProtocolпјҢи¶…ж–Үжң¬дј иҫ“еҚҸи®®пјү:жҳҜдёҖдёӘз”ЁдәҺдј иҫ“и¶…еӘ’дҪ“ж–ҮжЎЈзҡ„еә”з”ЁеұӮеҚҸи®®пјҢдё»иҰҒз”ЁдәҺWebжөҸи§ҲеҷЁдёҺWebжңҚеҠЎеҷЁд№Ӣй—ҙзҡ„йҖҡдҝЎгҖӮе°Ҫз®ЎжңҖеҲқи®ҫи®ЎжҳҜдёәдәҶж”ҜжҢҒзҪ‘йЎөжөҸи§ҲпјҢдҪҶHTTPд№ҹиў«е№ҝжіӣеә”з”ЁдәҺжһ„е»әRESTful APIзӯүжңҚеҠЎжҺҘеҸЈ

HTTPдёҺHTTPS

  • HTTP жҳҜи¶…вҪӮжң¬дј иҫ“еҚҸи®®пјҢдҝЎжҒҜжҳҜжҳҺвҪӮдј иҫ“пјҢеӯҳеңЁе®үе…Ёв»ӣйҷ©зҡ„й—®йўҳгҖӮHTTPS еҲҷи§ЈеҶі HTTP дёҚе®үе…Ёзҡ„зјә йҷ·пјҢеңЁ TCP е’Ң HTTP вҪ№з»ңеұӮд№Ӣй—ҙеҠ вјҠдәҶ SSL/TLS е®үе…ЁеҚҸи®®пјҢдҪҝеҫ—жҠҘвҪӮиғҪеӨҹеҠ еҜҶдј иҫ“гҖӮ
  • HTTP иҝһжҺҘе»әвҪҙзӣёеҜ№з®ҖеҚ•пјҢ TCP дёүж¬ЎжҸЎвјҝд№ӢеҗҺдҫҝеҸҜиҝӣвҫҸ HTTP зҡ„жҠҘвҪӮдј иҫ“гҖӮвҪҪ HTTPS еңЁ TCP дёүж¬Ў жҸЎвјҝд№ӢеҗҺпјҢиҝҳйңҖиҝӣвҫҸ SSL/TLS зҡ„жҸЎвјҝиҝҮзЁӢпјҢжүҚеҸҜиҝӣвјҠеҠ еҜҶжҠҘвҪӮдј иҫ“гҖӮ
  • дёӨиҖ…зҡ„й»ҳи®Өз«ҜвјқдёҚвјҖж ·пјҢHTTP й»ҳи®Өз«ҜвјқеҸ·жҳҜ 80пјҢHTTPS й»ҳи®Өз«ҜвјқеҸ·жҳҜ 443гҖӮ
  • HTTPS еҚҸи®®йңҖиҰҒеҗ‘ CAпјҲиҜҒд№ҰжқғеЁҒжңәжһ„пјүз”іиҜ·ж•°еӯ—иҜҒд№ҰпјҢжқҘдҝқиҜҒжңҚеҠЎеҷЁзҡ„иә«д»ҪжҳҜеҸҜдҝЎзҡ„

HTTPS еңЁ HTTP дёҺ TCP еұӮд№Ӣй—ҙеҠ вјҠдәҶ SSL/TLS еҚҸи®®пјҢеҸҜд»Ҙи§ЈеҶіе®үе…Ёв»ӣйҷ©пјҡ дҝЎжҒҜеҠ еҜҶпјҡдәӨдә’дҝЎжҒҜвҪҶжі•иў«зӘғеҸ–пјҢдҪҶдҪ зҡ„еҸ·дјҡеӣ дёәгҖҢвҫғиә«еҝҳи®°гҖҚиҙҰеҸ·вҪҪжІЎгҖӮ ж ЎйӘҢжңәеҲ¶пјҡвҪҶжі•зҜЎж”№йҖҡдҝЎеҶ…е®№пјҢзҜЎж”№дәҶе°ұдёҚиғҪжӯЈеёёжҳҫзӨәпјҢдҪҶзҷҫеәҰгҖҢз«һд»·жҺ’еҗҚгҖҚдҫқз„¶еҸҜд»Ҙжҗңзҙўеһғеңҫвјҙ е‘ҠгҖӮ иә«д»ҪиҜҒд№ҰпјҡиҜҒжҳҺж·ҳе®қжҳҜзңҹзҡ„ж·ҳе®қвҪ№пјҢдҪҶдҪ зҡ„й’ұиҝҳжҳҜдјҡеӣ дёәгҖҢеүҒвјҝгҖҚвҪҪжІЎгҖӮ
