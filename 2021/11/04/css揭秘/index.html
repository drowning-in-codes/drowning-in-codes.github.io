如果不给元素指定一个具体的height,它就会自动适应内容的高度

我们希望文字与图片一样宽,而且居中.

需要解决的问题,元素宽度由内部因素而不是外部因素决定.

减少媒体查询的使用

  1. 使用百分比的长度来取代固定长度.或者尝试使用与视口相关的单位(vw,vh,vmin,vmax)

  2. 较大分辨率得到固定宽度时使用max-width

  3. 为替换元素(img,object,video,frame)设置max-width,值为100%

  4. 如果背景图片需要完整铺满,使用cover

  5. 当图片等以行列式进行布局时,让视口宽度决定列的数量.弹性盒布局或者inlineblock加上文本换行做到

  6. 使用多列文本时指定列宽 column-width在较小屏幕上显示为单列布局
